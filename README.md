# Douban2Rargb (修改自Douban2Piratebay)
在豆瓣电影页面里添加一个指向Rargb的链接 
## Depends on
请确认已安装 [tampermonkey](http://tampermonkey.net/) 或 [greasemonkey](http://www.greasespot.net/) 插件
